## Tech stack
1. React with TypeScript as the backbone
2. Tailwind CSS to make things look good
3. GSAP for everything that moves
4. Deployment via GitHub Pages

# Technical documentation

## Prerequisites
- React.js: ^17.x
- Node.js: ^16.x

## Run commands
```shell
# Run at http://localhost:3000/
npm install
npm start
```

## Deploy commands
```shell
npm run deploy
```
